# tcc-propensao-emprestimo

# Modelagem Preditiva de Propensão à Contratação de Empréstimos
Este repositório contém o desenvolvimento do meu Trabalho de Conclusão de Curso (TCC) para o MBA em Data Science e Analytics da USP/ESALQ.

## Objetivo do Projeto
O objetivo principal é desenvolver e comparar modelos de Machine Learning para prever a propensão de um cliente contratar um empréstimo pessoal, subsidiando a otimização de estratégias de segmentação de público em instituições financeiras.

## Modelos Comparados
Nesta primeira versão, foram implementados os seguintes modelos:
- Regressão Logística (Baseline)
- Random Forest
- XGBoost
- LightGBM

## Estrutura do Repositório
- `Modelagem_Propensao_AndressaSouza.ipynb`: Notebook principal com todo o código.
- `graficos/`: Pasta contendo os gráficos exportados.
